# RP-discordbot
## РП бот для дискорда
### Инструкция по установке
**Скачайте два модуля**
Windows:
```
pip install discord.py
```
pip install colorama
```
MacOS/Linux:
```
pip3 install discord.py
```
pip3 install colorama
```
**Установка кода**
```
git clone https://github.com/0xSn1kky/RP-discordbot.git
```
